Favourite most recent project

Here you can see my favourite recent project

Marble Madness(unity remake)

This project was a University assignment of which we were given the barebones and had to create the rest ourselves. This project had to be created in 4 weeks, so I didn't have too much time to perfect or do anything crazy with it.'

This is my favourite most recent project because it was my first proper project back on unity since 2021. It was a pleasure creating this game.

The game objects/mechanics I created

My favourite game mechanic I created during this project was working doors. They were such a big part of this project for me because I have never created operating doors before and I love how they operate. To open the door, the player was first required to find a hidden colour change pad which acted like a key for the door.

Below is a list of other objects/mechanics I made during this project.

  • Jump pad.
  • Teleportation pad.
  • Bouncing pegs.
  • Enemy AI.
  • Moving platforms.
